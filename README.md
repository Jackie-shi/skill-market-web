# 🛒 Skill Market

AI Skills marketplace — discover, share & trade Skills for Claude Code, OpenClaw and beyond.

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Package Manager:** pnpm
- **Deployment:** Vercel

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/Jackie-shi/skill-market-web)

## CI/CD

GitHub Actions runs lint + build on every push to `main` and on PRs.
Vercel auto-deploys on push to `main` once connected.
